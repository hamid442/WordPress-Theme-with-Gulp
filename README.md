# WordPress Theme with Gulp
a simple WordPress theme with gulp
This theme use wpelixir.
#Requerment: 
	- npm
	- gulp

# Install

	1- download theme.
	2- copy to 'wp-content\themes\ThemeName'.
	3- in terminal go to `wp-content\themes\ThemeName` and run:
		```javascript
			npm install
		```
#Usage Gulp:
	- Edit "gulpfile.js".
	- run "gulp".
